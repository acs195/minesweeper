# Backend

## Run app
With Uvicorn:
```
uvicorn app/main:app --reload
```

Straigth with Python
```
python app/main.py
```

---

## Run tests
```
pytest -v
```

---

## Deploy to Google App Engine
```
gcloud app deploy
```

It is hosted on: https://cloudaleapi.ue.r.appspot.com
